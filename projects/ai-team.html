AI Team

A personal AI infrastructure that runs Claude, Gemini, and Codex as a collaborative team. Rather than prompting one model at a time, AI Team orchestrates structured debates and brainstorms across all three models simultaneously, with each assigned a distinct role: steelman, fatal flaw, and blind spot.

The system maintains a shared knowledge base that grows over time, giving the models persistent memory of past sessions and context about ongoing work. A local web dashboard provides access to team debates, brainstorms, one-on-one conversations with individual models, and a knowledge base editor.

How it works

  • Debate mode — adversarial two-round sessions where models challenge each other's positions
  • Brainstorm mode — generative sessions synthesized into a single consolidated output
  • 1-on-1 chat — direct conversation with any model, with full session history as context
  • Knowledge base — structured markdown files that accumulate insights across all sessions

Stack

  • Python / Flask backend with Server-Sent Events for live streaming
  • Claude, Gemini, and Codex CLIs orchestrated via subprocess
  • Vanilla JS frontend, runs locally at localhost:5001
  • macOS launch agent for always-on background service
